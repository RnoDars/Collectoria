### 3. Commande Docker Obsolète (Gravité : MOYENNE)

#### Problème

Scripts utilisaient `docker-compose` (obsolète) au lieu de `docker compose` (moderne).

```bash
# ❌ Obsolète (Compose V1)
docker-compose -f docker-compose.prod.yml up -d

# ✅ Moderne (Compose V2, intégré Docker CLI)
docker compose -f docker-compose.prod.yml up -d
```

**Historique** :
- Docker Compose V1 : standalone (`docker-compose`)
- Docker Compose V2 : plugin Docker CLI (`docker compose`)
- V1 deprecated depuis 2021
- V1 EOL juillet 2023

#### Impact

Scripts ne fonctionnaient pas sur systèmes modernes (Docker CLI ≥ 20.10.13).

#### Cause Racine

Pas de **checklist environnement cible** avant écriture scripts :
- Version Docker installée ?
- Commandes disponibles ?
- Extensions/plugins requis ?

### 4. Health Checks Inaccessibles (Gravité : ÉLEVÉE)

#### Problème

Health checks tentaient de joindre `localhost:8080` depuis l'**hôte Docker**, alors que le port n'est pas exposé.

```bash
# ❌ Depuis hôte (échoue, port non exposé)

# ✅ Depuis container (fonctionne)

**Solution implémentée** : Utiliser `docker exec` pour checker depuis l'intérieur du container.

#### Cause Racine

Pas de compréhension du **réseau Docker** :
- Ports exposés vs ports internes
- Réseau bridge vs host
- Accès host → container vs container → container

### 5. Cache Docker Ignoré (Gravité : MOYENNE)

#### Problème

Le script `cleanup.sh --aggressive` était marqué comme "nettoyage complet" mais **ignorait 2GB de cache Docker build**.

```bash
# Avant fix
docker system df
TYPE            TOTAL    SIZE
Build Cache     45       2.05GB  # ← Ignoré !

# Après fix (commit 3a449f4 + 421c21c)
docker builder prune -a -f  # Ajouté
```

**Tentative de détection (échouée)** :
```bash
# ❌ Logique fragile avec sed + bc
cache_size=$(docker system df | sed -n '4p' | awk '{print $4}' | sed 's/GB//')
if (( $(echo "$cache_size > 0.5" | bc -l) )); then
    # Cleanup
fi

# Problème : si format output change, silent failure
```

#### Cause Racine

**Tests unitaires absents pour logique de parsing**.

Le parsing de sortie Docker est fragile et doit être testé avec différents formats :
- Build cache vide
- Build cache en MB
- Build cache en GB
- Locale différente (virgule vs point)

### Étape 3 : Validation Références (OBLIGATOIRE)

**Checklist validation références** :
- [ ] Noms services Docker : Vérifiés contre `docker-compose.prod.yml`
- [ ] Noms containers : Vérifiés contre convention `collectoria-[service]-[env]`

**Script d'aide** : `scripts/lib/validate-script.sh`
```bash
#!/bin/bash
# Valide un script de déploiement

script_path="$1"

echo "Validation de $script_path..."

# 1. Shellcheck
shellcheck "$script_path" || exit 1

# 2. Syntax check
bash -n "$script_path" || exit 1

# 3. Validation noms services
grep -o 'SERVICE_NAME="[^"]*"' "$script_path" | while read line; do
    service=$(echo "$line" | cut -d'"' -f2)
    if ! grep -q "^  $service:" docker-compose.prod.yml; then
        echo "ERROR: Service '$service' not found in docker-compose.prod.yml"
        exit 1
    fi
done

# 4. Validation fonctions
grep -oP '\b\w+\s*\(' "$script_path" | sed 's/($//' | while read func; do
    if ! grep -q "^$func()" scripts/lib/*.sh; then
        echo "WARNING: Function '$func' not found in libraries"
    fi
done

echo "✅ Validation passed"
```

### Étape 4 : Test Production (Dry-run OBLIGATOIRE)

**Avant déploiement en production** :
```bash
# 1. Dry-run sur serveur prod (prévisualisation)
./deploy-backend.sh --dry-run

# 2. Review output : Toutes les actions sont correctes ?

# 3. Exécution réelle
./deploy-backend.sh
```

**Règle absolue** : JAMAIS de première exécution directe en production sans dry-run.
```

#### R1.3 - Shellcheck Obligatoire (Linter)

**Problème** : Aucun linting des scripts Bash.

**Solution** : Intégrer shellcheck dans workflow.

**Installation** :
```bash
# Ubuntu/Debian

# macOS
brew install shellcheck
```

**Usage** :
```bash
# Lint script
shellcheck scripts/deploy/deploy-backend.sh

# Lint tous les scripts
find scripts/ -name "*.sh" -exec shellcheck {} \;
```

**Intégration Pre-commit Hook** :
```bash
#!/bin/bash
# .git/hooks/pre-commit

# Shellcheck on staged .sh files
staged_scripts=$(git diff --cached --name-only --diff-filter=ACM | grep '\.sh$')

if [[ -n "$staged_scripts" ]]; then
    echo "Running shellcheck on staged scripts..."
    for script in $staged_scripts; do
        if ! shellcheck "$script"; then
            echo "❌ Shellcheck failed for $script"
            exit 1
        fi
    done
    echo "✅ Shellcheck passed"
fi
```

**Règle** : Aucun commit de script Bash sans shellcheck passing.

#### R2.3 - Script de Validation Automatique

**Problème** : Validation manuelle fastidieuse.

**Solution** : Script `scripts/lib/validate-script.sh` qui automatise validations.

```bash
#!/bin/bash
#
# validate-script.sh - Validate Bash deployment script
#
# Usage:
#   ./validate-script.sh <script-path>
#
# Example:
#   ./validate-script.sh ../deploy/deploy-backend.sh

set -e

SCRIPT_PATH="$1"

if [[ -z "$SCRIPT_PATH" ]]; then
    echo "Usage: $0 <script-path>"
    exit 1
fi

if [[ ! -f "$SCRIPT_PATH" ]]; then
    echo "ERROR: Script not found: $SCRIPT_PATH"
    exit 1
fi

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
PROJECT_ROOT="$SCRIPT_DIR/../.."

echo "========================================="
echo "Validating Script: $(basename "$SCRIPT_PATH")"
echo "========================================="
echo ""

# Counter
ERRORS=0
WARNINGS=0

# 1. Shellcheck
echo "1. Running shellcheck..."
if command -v shellcheck >/dev/null 2>&1; then
    if shellcheck "$SCRIPT_PATH"; then
        echo "   ✅ Shellcheck passed"
    else
        echo "   ❌ Shellcheck failed"
        ((ERRORS++))
    fi
else
    echo "   ⚠️  Shellcheck not installed (skip)"
    ((WARNINGS++))
fi
echo ""

# 2. Syntax check
echo "2. Syntax check..."
if bash -n "$SCRIPT_PATH" 2>/dev/null; then
    echo "   ✅ Syntax OK"
else
    echo "   ❌ Syntax error detected"
    bash -n "$SCRIPT_PATH"
    ((ERRORS++))
fi
echo ""

# 3. Check set -e
echo "3. Checking 'set -e'..."
if grep -q "^set -e" "$SCRIPT_PATH"; then
    echo "   ✅ 'set -e' present"
else
    echo "   ⚠️  'set -e' missing (should exit on error)"
    ((WARNINGS++))
fi
echo ""

# 4. Check --dry-run support
echo "4. Checking --dry-run support..."
if grep -q "DRY_RUN" "$SCRIPT_PATH"; then
    echo "   ✅ --dry-run support detected"
else
    echo "   ⚠️  --dry-run not implemented"
    ((WARNINGS++))
fi
echo ""

# 5. Validate service names
echo "5. Validating Docker service names..."
SERVICE_NAMES=$(grep -oP 'SERVICE_NAME="\K[^"]+' "$SCRIPT_PATH" || true)
if [[ -n "$SERVICE_NAMES" ]]; then
    for service in $SERVICE_NAMES; do
        if grep -q "^  $service:" "$PROJECT_ROOT/docker-compose.prod.yml" 2>/dev/null; then
            echo "   ✅ Service '$service' found in docker-compose.prod.yml"
        else
            echo "   ❌ Service '$service' NOT FOUND in docker-compose.prod.yml"
            ((ERRORS++))
        fi
    done
else
    echo "   ℹ️  No SERVICE_NAME defined (skip)"
fi
echo ""

# 6. Validate container names
echo "6. Validating Docker container names..."
CONTAINER_NAMES=$(grep -oP 'CONTAINER_NAME="\K[^"]+' "$SCRIPT_PATH" || true)
if [[ -n "$CONTAINER_NAMES" ]]; then
    for container in $CONTAINER_NAMES; do
        # Check naming convention: collectoria-[service]-[env]
        if [[ "$container" =~ ^collectoria-[a-z-]+-prod$ ]]; then
            echo "   ✅ Container '$container' follows convention"
        else
            echo "   ⚠️  Container '$container' doesn't follow convention 'collectoria-[service]-prod'"
            ((WARNINGS++))
        fi
    done
else
    echo "   ℹ️  No CONTAINER_NAME defined (skip)"
fi
echo ""

# 7. Validate function calls
echo "7. Validating function calls..."
FUNCTIONS=$(grep -oP '\b[a-z_]+\s*\(' "$SCRIPT_PATH" | sed 's/($//' | sort -u || true)
if [[ -n "$FUNCTIONS" ]]; then
    for func in $FUNCTIONS; do
        # Skip built-in bash commands
        if [[ "$func" =~ ^(echo|printf|read|cd|exit|return|local|eval|shift|grep|sed|awk|sort|cut|wc|find)$ ]]; then
            continue
        fi
        
        # Check if function exists in libraries
        if grep -q "^${func}()" "$SCRIPT_DIR"/*.sh 2>/dev/null; then
            echo "   ✅ Function '$func' found in libraries"
        else
            echo "   ⚠️  Function '$func' not found in libraries (may be defined in script)"
            ((WARNINGS++))
        fi
    done
else
    echo "   ℹ️  No custom functions called"
fi
echo ""

# 8. Check header documentation
echo "8. Checking header documentation..."
if head -20 "$SCRIPT_PATH" | grep -q "# Description:"; then
    echo "   ✅ Header description present"
else
    echo "   ⚠️  Header description missing"
    ((WARNINGS++))
fi

if head -20 "$SCRIPT_PATH" | grep -q "# Usage:"; then
    echo "   ✅ Usage documentation present"
else
    echo "   ⚠️  Usage documentation missing"
    ((WARNINGS++))
fi
echo ""

# 9. Check docker-compose vs docker compose
echo "9. Checking Docker Compose command..."
if grep -q "docker-compose " "$SCRIPT_PATH"; then
    echo "   ❌ 'docker-compose' (V1, obsolete) detected"
    echo "      Use 'docker compose' (V2) instead"
    ((ERRORS++))
elif grep -q "docker compose " "$SCRIPT_PATH"; then
    echo "   ✅ 'docker compose' (V2) used"
else
    echo "   ℹ️  No docker compose commands detected"
fi
echo ""

# Summary
echo "========================================="
echo "Validation Summary"
echo "========================================="
echo "Errors:   $ERRORS"
echo "Warnings: $WARNINGS"
echo ""

if [[ $ERRORS -gt 0 ]]; then
    echo "❌ REJECTED: Fix errors before committing"
    exit 1
elif [[ $WARNINGS -gt 0 ]]; then
    echo "⚠️  WARNINGS: Review before committing"
    exit 0
else
    echo "✅ APPROVED: Script ready to commit"
    exit 0
fi
```

**Usage** :
```bash

./scripts/lib/validate-script.sh scripts/deploy/deploy-backend.sh

# Valider tous les scripts
find scripts/ -name "*.sh" -not -path "*/lib/*" -exec ./scripts/lib/validate-script.sh {} \;
```

**Intégration Pre-commit Hook** :
```bash

if [[ -n "$staged_scripts" ]]; then
    for script in $staged_scripts; do
        if ! ./scripts/lib/validate-script.sh "$script"; then
            echo "❌ Validation failed for $script"
            exit 1
        fi
    done
fi
```
